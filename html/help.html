Help and FAQs
What is BedStats made with?
+
BedStats is built on electron.js, a framework that allows for a mixture of Node.js and WebDev. The windows are displayed with HTML5/CSS/Chrome V8, while the background process is run with Node.
Why is BedStats so laggy?
+
Since BedStats is built on electron.js, a framework that comes packaged with Chromium, this application is required to be bundled with the Chromium engine. As such, you should treat it more like a Discord tab or Chrome window, as opposed to a .NET or C application.
Is BedStats allowed on the Hypixel Network?
+
Theoretically, since it doesn't directly hook into the client, it should be. However, like many other auto-stat trackers, developers treat apps like these as "use as your own risk". I'm not responsible for what you do with this application.
What is a Hypixel API Key?
+
A hypixel API key is what allows the program to access the Hypixel API, or application programming interface. Calling on the Hypixel API allows the tracker to get a player's stats, and as such, a key is needed. Keep in mind that normal keys are rate-limited to 120 requests per minute, and as such may not work for large operations (e.g. simultaneously searching stats for a large group of friends).
Why is this program not fully functional?
+
BedStats is not a fully complete application, as development only started in late February. More features are to come as monthly updates are pushed out.
